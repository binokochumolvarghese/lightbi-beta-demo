**Lightbi** is built on the shoulders of many excellent open source tools. Deep thanks to every contributor who has shared their work with the community.

### Framework & Hosting

- [Hugo](https://gohugo.io/) — the static site generator that powers this theme. Incredibly fast, flexible, and a joy to build with.
- [Netlify](https://www.netlify.com/) — hosting, continuous deployment, and free SSL via [Let's Encrypt](https://letsencrypt.org/).

### Design & UI

- [Bootstrap 5](https://getbootstrap.com/) — responsive grid, components, and utility classes.
- [Bootstrap Icons](https://icons.getbootstrap.com/) — clean SVG icon set used throughout the theme.
- [Inter](https://rsms.me/inter/) — the main body typeface, a beautiful variable sans-serif by [Rasmus Andersson](https://rsms.me/).
- [Playfair Display](https://fonts.google.com/specimen/Playfair+Display) — the heading typeface, a transitional serif by Claus Eggers Sørensen.

### Search

- [Pagefind](https://pagefind.app/) — fully static search with zero runtime dependencies. Replaces the previous Lunr.js integration.

### Media

- [GLightbox](https://biati-digital.github.io/glightbox/) — accessible lightbox for images in posts.
- [Medium Zoom](https://medium-zoom.francoischalifour.com/) — smooth image zoom on click.
- [Unsplash](https://unsplash.com/) — source of the demo cover images.

### Tooling

- [Claude Code](https://claude.ai/code) — assisted with theme development, dark mode implementation, and layout improvements.
